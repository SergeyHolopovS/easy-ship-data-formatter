# Сайт-форматтер для проектов кораблей

Простой сайт для форматирования данных с сайтов проектов кораблей в необходимый формат

## Техническое задани

Разработать сайт, преобразующий данные пользовательского ввода в необходимый формат данных JSON

### Модель проекта:
- name - название проекта (`string`)
- dictionary - словарь основных характеристик (`string - string`)
- arrays - словарь дополнительных характеристик (`string: string[]`)
- other - список прочих характеристик (`string[]`)

### Пример результата:
```json
{
    "name": "Название проекта",
    "dictionary": {
        "ключ1": "значение1",
        "ключ2": "значение2"
    },
    "arrays": {
        "ключ3": [
            "значение3",
            "значение4"
        ],
        "ключ4": [
            "значение5",
            "значение6"
        ],
    },
    "other": [
        "значение7",
        "значение8"
    ]
}
```
